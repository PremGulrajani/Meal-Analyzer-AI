# MealAnalyzer AI — Prem's AI Nutritional Coach

MealAnalyzer AI is a cloud-based conversational nutrition assistant designed to make tracking nutrition accurate, low-effort, and sustainable.

Most nutrition apps rely on manual logging and raw numbers. MealAnalyzer AI instead allows users to describe meals naturally, while AI reasoning and structured nutrition data provide contextual insights and daily tracking automatically.

## Tech Stack

1. Google Cloud Run (Compute)
2. Vertex AI Gemini 2.0 Flash Lite (AI Reasoning)
3. Cloud Firestore (Data Persistence & Caching)
4. Cloud Speech-to-Text (Voice Processing)
5. USDA FoodData Central API (External Data Tool)
6. Cloud Secret Manager (Security and Governance)
7. Google Cloud IAM (Security & Governance)
